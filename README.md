## How to run

1. `yarn install` or `npm install`
2. rename `.env.example` as `.env`
3. `docker compose up`
4. check on `http://localhost:3000/health`


## Swagger link
`http://localhost:3000/api`

### Use Swagger
1. Run `auth/signIn` with example value
2. Copy `accessToken` and put in `Authorize` tab
